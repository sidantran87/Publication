This repository contains the source code and reports for two intensive research projects on Predictive Analytics using Machine Learning and Deep Learning in the fields of Finance and Digital Services. Both studies focus on leveraging time-series and sequential data structures to optimize prediction performance and model interpretability.

- Enhancing Default Prediction in P2P Lending with Deep Model Fusion: Focuses on predicting bad debt risk in Peer-to-Peer lending by combining static attribute data with sequential transactional history.
- Flexible Approach to Churn Prediction using Time Series Classification: Proposes the MiniRocket-SHAP model to predict customer churn for online music services, identifying both the timing and the causes of departure.

- Language: Python
- Deep Learning: LSTM, Transformers, MLP, Bi-LSTM.

- Data Processing: SMOTE (imbalance handling), Feature Engineering for time series.
